### Ingress-NGINX Configurations

This repo holds several configuration options for `ingress-nginx`. *NOTE* as of March 2026, the project will be deprecated and no longer maintaned or developed. See [Ingress NGINX Retirement: What You Need to Know](https://kubernetes.io/blog/2025/11/11/ingress-nginx-retirement/) from the upstream documentation for more information. This repo is keeping the data for historical and archival purposes. Your organization, lab/development environment should consider moving to another ingress controller or Gateway API in the near future.
