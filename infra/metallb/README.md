### Metallb 

MetalLB is a load-balancer implementation for bare metal Kubernetes clusters, using standard routing protocols. It supports both Layer2 and Layer3 as well as ipv4 and ipv6.

Subdirectories: 
* l2adv - lab experiment with L2 Advertisements (ARP).
* l3adv - lab experiment with L3 Advertisements (BGP).

The latest version includes many important updates and security fixes, see the relaese notes below.
* https://metallb.universe.tf/release-notes/#version-0-15-3
