## Kata Containers Deployment

Kata Containers is an open source community working to build a secure container runtime with lightweight virtual machines (VM's) that feel and perform like standard Linux containers, but provide stronger workload isolation using hardware virtualization technology as a second layer of defense.

This oci helm chart includes default values that operate with RKE2 and K3S. SELinux host-level support is currently working well along with the standard `container-selinux` distribution package.

See the [Kata Containers Docs](https://kata-containers.github.io/kata-containers/installation/) for more install information.
